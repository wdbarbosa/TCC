 Software do Cursinho Preparatório Primeiro de Maio UNESP

 Descrição
Projeto focado em auxiliar alunos que participam do Cursinho Primeiro de Maio, fornecendo uma plataforma que facilita a organização e o acompanhamento das atividades.

 Tecnologias
- PHP  
- Laravel  
- Breeze  
- Composer  
- PGadmin  
- Node.js

 Instalação
Siga os passos abaixo para instalar o projeto em sua máquina:

1. Inicialize o repositório Git:  
	git init
2. Adicione o repositório remoto:
	git remote add origin https://github.com/wdbarbosa/TCC.git
3.Baixe os arquivos do repositório:
	git pull origin main
4.Atualize as dependências do Composer:
	composer update

Como Usar
1.Compile os arquivos frontend:
	npm run dev
2.Inicie o servidor de desenvolvimento do Laravel:
	php artisan serve
